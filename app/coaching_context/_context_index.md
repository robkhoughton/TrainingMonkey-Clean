# COACHING CONTEXT — ORIENTATION

This block is a **state-gated subset** of the coaching context library. Not all topic files are present — gating depends on readiness state and race proximity.

**Absence of a topic does not mean no constraints apply.** If a topic is not addressed in this block, treat it as out of scope for today's context and default to conservative thresholds. Do not infer that uncovered topics are unconstrained.

**Conflict resolution priority:**
1. `readiness.md` — overrides all intensity and scheduling guidance when present; a non-GREEN readiness state shifts every threshold downward.
2. Specific topic files govern over general files on their subject (e.g., `fueling.md` governs carb timing even if `intensity_zones.md` touches it).
3. `intensity_zones.md` — foundational defaults when no specific file addresses a topic.

**Each directive stands alone.** Do not expect cross-references between files to resolve — a file that names another may be referring to one that was gated out.
