What is Enhanced ACWR Calculation?

The enhanced ACWR calculation uses exponential decay weighting instead of simple averaging for chronic load calculation. This provides more accurate training load monitoring by giving more weight to recent activities.

  • Standard ACWR: Uses 28-day simple average for chronic load
  • Enhanced ACWR: Uses configurable period (28-90 days) with exponential decay weighting
  • Benefits: More accurate training load monitoring, better injury prevention
