Step-by-Step Deployment from React to Cloud Run:

## 1. Rebuild React App
cmd
cd C:\Users\robho\Documents\TrainingMonkey-Clean\frontend
npm run build

## 2. Copy to Deployment
cmd
cd..
xcopy frontend\build\* app\build\ /E /Y
xcopy frontend\build\static\* app\static\ /E /Y
copy frontend\build\training-monkey-runner.webp app\static\training-monkey-runner.webp

## 3. Deployment
cmd
cd app
deploy_strava_simple.bat

## Alternative: Use Automated Scripts
scripts\deploy.bat          # For production deployment
scripts\deploy_local.bat    # For local development
